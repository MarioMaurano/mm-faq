01 The Pythagorean System

The Pythagorean system is an axiomatic tuning system — every pitch is derived from a single indisputable rule: the arithmetic fraction 3/2 (the pure perfect fifth). No approximation is ever made.

Every one of the 35 pitches in Music Motion is expressed as an exact arithmetic fraction of powers of 2 and powers of 3 — written in the form 3ⁿ / 2ᵐ. This is the only mathematical operation used in the Pythagorean context.

The system does not use any ratio derived from roots of 2. It does not use the circle of fifths as a conceptual tool. It does not reference cents, quarter-tones, or half-steps. These belong exclusively to the tempered context.

The 7 Fundamentals are the diatonic degrees of the major scale. Each ratio is derived exclusively by multiplying or dividing 3/2 and 9/8, then reducing by factors of 2 to remain within the octave.

The mathematically exact gap between enharmonic notes — such as C♯ and D♭ — when the Pythagorean system is applied strictly:

531441 / 524288 = 3¹² / 2¹⁹

Equal temperament erases this gap by approximation. In Music Motion it remains audible. Example: D♯ = 1184.135 Hz · E♭ = 1165.845 Hz — two physically distinct zones on the mmk keyboard.

The Axiomatic System contains all 35 Pythagorean pitches in pure mathematical order, without tonal context. The Tonal System anchors on a chosen key — its 7 diatonic degrees remain pure Pythagorean; alterations (♭ ♮ ♯) shift each degree by one diatonic semitone 256/243 without crossing adjacent degrees.

7 keys (F C G D A E B) × 3 alterations = 21 tonal contexts, all expressed as fractions of 2 and 3.

02 The mmk Keyboard

A free, browser-based instrument making 35 mathematically exact Pythagorean pitches per octave directly playable — no installation, no server, no dependencies. Open the HTML file in any modern browser.

Conceived by Mario Maurano, contrabassist and former principal bass of the BRT Philharmonic Orchestra (18 years) and Orchestre Philharmonique de Liège (17 years), from decades of first-hand experience with pure intonation.

Standard keyboards merge enharmonic notes into one key — an approximation called equal temperament. The mmk keeps each enharmonic pair as two physically separate zones with distinct frequencies: D♯ ≠ E♭, F𝄪 ≠ G, B𝄫 ≠ A — restored as exact arithmetic fractions of 2 and 3.

White keys (except A4): upper 4/5 = pure Pythagorean pitch · lower 1/5 = equal-tempered reference (shown in cyan). A4 is undivided — the universal 440 Hz reference.

Black keys (and E→F, B→C positions): top 2/5 = ♭♭ / ♯♯ double accidentals · middle 2/5 = ♭ / ♯ single accidentals · bottom 1/5 = equal-tempered reference. The E→F and B→C keys are ivory-coloured to mark their special axiomatic status.

All instruments are synthesized in Web Audio API — zero external dependencies, up to 6 simultaneous voices. Families: Synthetic · Strings · Woodwinds · Brass · Percussion · Keyboard · Plucked & Folk.

Controls: System (Axiomatic / Tonal) · Key (F C G D A E B) · Alteration (♭ ♮ ♯) · Octave · Tuning A= (415 · 430 · 432 · 440 · 442 · 466 Hz) · Theme (4 options) · 12 display languages.

Press ⏺ to record — every note is captured with its exact Hz frequency as a 3ⁿ / 2ᵐ ratio, not a MIDI note number. Press ⏹ and save as .mmf (Music Motion Format, JSON-based). Each file stores: exact frequency · OO/O1 status · deduced root · motion data (glissandi, vibrato) · performance data (velocity, timing).

03 The mm Sequencer

Accepts any MIDI (.mid), MusicXML (.xml / .mxl), or .mmf file and converts it to pure Pythagorean intonation in real time. A single self-contained HTML file — no installation, no server.

Core workflow — Bach to the Future: load any MIDI → hear it in Pythagorean tuning → save as .mmf → perform live on the keyboard.

PipelineNameContent
1PerformanceRaw note, velocity, timing — unchanged
2MotionOriginal pitch bend — glissandi, vibrato — preserved exactly
3AxiomaticPythagorean correction: exact 3ⁿ / 2ᵐ ratios · OO/O1 status

An optional AI layer that automatically identifies the tonal context of each passage, assigns OO/O1 status per note, and determines the Pythagorean sensitization level. Connects via your own Anthropic API key — stored only in your browser, never shared. The Sequencer is fully functional without it.

04 Tempered Systems

Equal temperament is also axiomatic — but its axiom is the n-th root of 2 (e.g. the 12th root for the standard 12-tone system). This generates irrational numbers — frequencies that cannot be expressed as exact fractions of whole numbers.

The Pythagorean system uses only arithmetic fractions of 2 and 3. Every pitch is exact.

Cents, quarter-tones, and half-steps as measurement units belong to the tempered context only. They are not used to describe or derive Pythagorean ratios in Music Motion.

05 Using the Tools

No installation and no server are required. Each tool is a single self-contained HTML file running entirely in Web Audio API — zero external dependencies. Note: Chrome requires a local HTTP server (python -m http.server 8080) due to its audio policy. Firefox and Safari open files directly.

The Music Motion basic keyboard is and will remain free — open to every student, musician, and researcher worldwide. The roadmap includes mmk Pro (commercial, full composition environment) and a physical mmk keyboard (commercial, native .mmf support). The project is crowdfunding-supported to preserve its mathematical independence.
